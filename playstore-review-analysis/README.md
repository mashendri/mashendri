# Google Play Store Review Analysis 📊

A sentiment analysis and NLP tool for Google Play Store app reviews. This project includes a data scraper, text preprocessing (NLP), and an interactive dashboard using Streamlit.

## ✨ Key Features
- **Automated Scraping**: Fetch reviews in real-time based on an App ID and a specific starting year.
- **Sentiment Analysis**: Classifies sentiment (Positive, Negative, Neutral) using a Lexicon-based approach, with advanced AI validation (IndoBERT).
- **Interactive Dashboard**: Visualizes weekly trends, sentiment distribution, and keyword Word Clouds.
- **Anomaly Detection**: Identifies contradictions between Play Store star ratings and the text sentiment.
- **Data Export**: Export the analysis results to CSV format.

## 🛠️ Requirements
- Python 3.9+
- Pip (Python Package Manager)

## 🚀 How to Run Locally

### 1. Environment Setup
Clone this repository or move to the project directory, then create a virtual environment:

```bash
# Create virtual environment
python3 -m venv venv

# Activate virtual environment
# For Mac/Linux:
source venv/bin/activate
# For Windows:
# venv\Scripts\activate
```

### 2. Install Dependencies
Install all required libraries:

```bash
pip install -r requirements.txt
```

### 3. Run the Analysis
You have three main components that can be run separately:

#### A. Scrape New Data
Run `scraper.py` to fetch the latest review data (by default, it fetches data from the previous year):
```bash
python3 scraper.py
```

#### B. Process Data (NLP & Sentiment)
Run `preprocess.py` to clean the text and analyze the sentiment from the CSV file generated by the scraper:
```bash
python3 preprocess.py
```

#### C. Run Streamlit Dashboard
Run the dashboard to see data visualizations:
```bash
streamlit run app.py
```
Once run, open your browser at `http://localhost:8501`.

## 📂 Project Structure
- `app.py`: Main file for the Streamlit dashboard.
- `scraper.py`: Script to fetch data from the Google Play Store.
- `preprocess.py`: A collection of functions to clean text and perform sentiment analysis.
- `requirements.txt`: List of required Python libraries.
- `playstore_reviews_*.csv`: Raw data file from scraping.
- `processed_playstore_reviews_*.csv`: Processed NLP data file.
